Pasos para Ejecutar:

1. montar el proyecto en NetBeans o ejecutar PruebaCerticamara.jar

2. digitar el número de filas

3. digitar el número de columnas

4. en el menú de opciones seleccionar 1 para agregar las rutas de la nueva posición y 2 para terminar el programa

5. si selecciona 1 permite digitar las rutas según la especificación  <Desplazamiento>,<Dirección> o  <Desplazamiento>,<Dirección>; <Desplazamiento>,<Dirección> ... presionar enter

6. se listan por consola el comando aplicado, y el punto final luego de cada desplazamiento

7. aparece nuevamente el menú para digitar más rutas.
